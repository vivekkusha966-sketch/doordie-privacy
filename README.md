  <h1>Privacy Policy for MyApplicationDoORDie</h1>
  <p>Effective Date: June 17, 2026</p>

  <h2>Core Mission</h2>
  <p>
    MyApplicationDoORDie is a productivity and focus tool designed to help users
    enter a "Flow State" (Zone Mode) by combining task management with aggressive
    focus enforcement. The app enforces focus by restricting access to distractions
    and keeping tasks visible through overlays and dynamic wallpapers.
  </p>

  <h2>Information We Collect</h2>
  <ul>
    <li>Task data you enter (titles, notes, reminders).</li>
    <li>Focus session history (time spent, streaks, scoreboard).</li>
    <li>Foreground app package names (via Accessibility Service) — used only for blocking distractions.</li>
  </ul>

  <h2>How We Use Information</h2>
  <ul>
    <li>To enforce focus by blocking selected apps during Focus Mode.</li>
    <li>To display your current tasks via HUD overlays and dynamic wallpapers.</li>
    <li>To provide tactical feedback (haptics, UI cues) when tasks are completed.</li>
  </ul>

  <h2>Data Storage and Protection</h2>
  <p>
    All data is stored locally on your device using Room Database and DataStore.
    Task data auto-deletes after 30 days.  
    No personal information is transmitted off the device.
  </p>

  <h2>Sensitive Permissions</h2>
  <ul>
    <li><strong>Accessibility Service:</strong> Used only to detect the foreground app for App Blocker and HUD features. No personal data is collected or shared.</li>
    <li><strong>System Alert Window:</strong> Used to render overlays (HUD, Quick Add). No personal data is collected.</li>
    <li><strong>Wallpaper Service:</strong> Used to display tasks on your home screen background. No personal data is collected.</li>
  </ul>

  <h2>Third-Party Sharing</h2>
  <p>
    We do not share your data with third parties.  
    The app does not use ads, analytics, or external tracking services.
  </p>
